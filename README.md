# veterinary-hospital
## 동물병원 예약제 시스템



## 개발기간 
- (목표) 24.01.15 ~ 24.01.31



## 개발 환경
- java17
- **IDE** : intelliJ
- **Framework** : Springboot(3.x)
- **Database** : Mysql(8.1.0)
- **ORM** : JPA

- 로그인 시 쿠키 및 세션 생성

#### 회원가입( 회원, 수의사 )
- 회원가입(완료)
- ID 중복 체크(완료)

#### 반려동물
- 반려동물 등록(완료)

#### 회원 마이페이지
- 회원 정보 변경
- 진료 기록 확인
- 반력동물 정보 변경

#### 수의사 마이페이지
- 수의사 정보 변경
- 오늘의 진료 일정 확인

#### 진료 예약
