# veterinary-hospital
## 동물병원 예약제 시스템



## 개발기간 
- (목표) 24.01.15 ~ 24.01.31



## 개발 환경
- java17
- **IDE** : intelliJ
- **Framework** : Springboot(3.x)
- **Database** : Mysql(8.1.0)
- **ORM** : JPA



## 주요기능
#### 로그인
- DB값 검증
- ID찾기, PWD찾기
- 로그인 시 쿠키 및 세션 생성

#### 회원가입
- ID 중복 체크

#### 마이페이지
- 회원정보 변경
- 진료 내역 확인

#### 진료 예약
- 진료 목적 작성
- 진료 날짜 지정
- 담당의 선택
- 예약 완료
